The action attribute defines the action to be preformed when the form is submitted.


Normally, the form data is sent to a web page on the server when the user clicks on the submit button.


In the example above, the form data is sent to a page on the server called "/action_page.php". This page containts a server-side script that handles the form data:


<form action="/action_page.php">


If the action attribute is omitted, the action is set to the current page.
