Click on the sun or on one of the planets to download its content.

Notice that the filename of the downloaded file will be saved as "sun.htm" instead of "information_about_the_sun.htm" for the sun area,"mercury.gif" instead of "merglobe.gif" for the mercury area and "venus.txt" instead of "information_about_the_planet_venus.txt" for the venus area.

Note: The download attribute is not supported in IE, Safari or Opera version 12 (and earlier).
