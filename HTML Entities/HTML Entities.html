Some characters are reserved in HTML.
If you use the less than (<<) or greater then (>>) signs in your text,the browser might mix them with tags.
Character entities are used to display reserved characters in HTML.
A character entity looks like this:
&entity_name;
OR
&38;#entity_number;
To display a less then sign (<) we must write &lt; or &#60;
Advantage of using an entity name: An entity name is easy to remember.
Disadvantage of using entity name: Browser may not support all entity names,but the support for numbers is good.
